== Description ==

Create an awesome pins for your image. It can be use for any highlighted points and dots on your image.

### Used

Copy your shortcode to view eg: [xolo_hotspot  id="YOUR_ID"]


### jQuery plugin include

- jQuery PowerTip [PowerTop Home](https://stevenbenner.github.io/jquery-powertip/)


== Installation ==

= Option 1. =

- Upload the entire `xolo-hotspot` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Hotspot" —> Click install.
- After plugin is downloaded, click active plugin

You will find 'Hotspot' menu in your WordPress admin panel.
